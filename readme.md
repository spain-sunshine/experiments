# TCP通信项目

## 项目简介
一个简单的TCP通信示例，包括一个服务器和一个客户端。

## 安装和设置
1. 确保已安装Python 3.x。
2. 克隆或下载此项目。

## 如何运行
1. 运行服务器: `python server.py`
2. 运行客户端: `python client.py`

## 示例输出
服务器启动，等待连接...
连接来自 ('127.0.0.1', SOME_PORT_NUMBER)
收到数据: 你好，我是客户端！

客户端输出:
收到服务器的响应: 你好，我是服务器！

## 技术细节
此项目基于TCP/IP协议实现简单的双向通信。
